LANGKAH AWAL

  1. Komputer dengan sistem operasi yang mendukung bahasa pemrograman C++.
  2. Perangkat lunak pemrograman (compiler/IDE), pada praktikum ini menggunakan Code::Blocks.
  3. Modul Praktikum.

PROSEDUR PRATIKUM 1

  1. Download installer di situs: https://www.codeblocks.org. Pilih installer sesuai sistem operasi komputer Anda. Disarankan mengunduh codeblocks-20.03mingw-setup.exe karena sudah menyertakan compiler standar yang siap digunakan.
  1. Setelah mengunduh, klik dua kali file installer untuk memulai instalasi. Ikuti wizard dengan menekan tombol "Next" hingga selesai pada pengaturan default.
  1. Atur compiler di menu Settings > Compiler > Toolchain executables > Auto-Detect > OK.

PROSEDUR PRATIKUM 2

  1. Klik menu File > New > Project, pilih "Console Application", lalu tekan "Go". Pilih bahasa C++ dan beri nama project sesuai keinginan.
  1. Klik "Finish" untuk konfigurasi default. Buka file main.cpp di jendela "Management" untuk mengedit kode.
